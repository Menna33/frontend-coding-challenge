# Front-end Coding Challenge

## Idea of the App 
It is a small webapp that lists the most starred Github repos that were created in the last 30 days. 
fetchs the sorted JSON data directly from the Github API (`https://api.github.com/). 

## Features
* The user lists the most starred Github repos that were created in the last 30 days. 
* The user sees the results as a list. One repository per row. 
* The user sees for each repo/row the following details :
  * Repository name
  * Repository description 
  * Number of stars for the repo. 
  * Number of issues for the repo.
  * Username and avatar of the owner. 
* When the user keeps scrolling, new results appear.


## Technologies  
* React
